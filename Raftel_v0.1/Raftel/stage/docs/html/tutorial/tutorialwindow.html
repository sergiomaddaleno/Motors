Opening a Black Window in Raftel Engine

In this tutorial, you will learn how to open a simple black window using the Raftel Engine. This is a fundamental skill necessary for any graphical application development.

Prerequisites

Before you begin, make sure you have Raftel Engine installed on your system. You should also have a basic understanding of C++ programming.

Step-by-Step Guide

Step 1: Include the Necessary Header

Start by including the Raftel window system header in your main program file:

#include "raftel/window.hpp"

Step 2: Initialize the Window System

Create an instance of the window system:

auto windowSystemOpt = Raftel::WindowSystem::make();

Step 3: Create the Window

Next, create a window using the window system:

auto windowOpt = Raftel::Window::make("WindowTest", *windowSystemOpt);
if (!windowOpt) {
  std::cerr << "Can't create window\n";
  return -1;
}

Step 4: Set the Window Context and Render Loop

Set the current context of the window for rendering and create a loop to keep the window open:

windowOpt->MakeContextCurrent();
while (!windowOpt->ShouldClose()) {
  windowOpt->clear();
  windowOpt->swapBuffers();
}

Conclusion

Now that you have successfully created a black window using Raftel Engine, you can begin to explore more complex graphical programming tasks. In future tutorials, we will cover how to handle user input and how to render basic 2D and 3D graphics.

Complete Code

Here is the complete code for this tutorial:

#include "raftel/window.hpp"

int main(void) {
  auto windowSystemOpt = Raftel::WindowSystem::make();
  auto windowOpt = Raftel::Window::make("WindowTest", *windowSystemOpt);
  if (!windowOpt) {
      std::cerr << "Can't create window\n";
      return -1;
  }

  windowOpt->MakeContextCurrent();
  while (!windowOpt->ShouldClose()) {
      windowOpt->clear();
      windowOpt->swapBuffers();
  }

  return 0;
}
