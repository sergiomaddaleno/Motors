Drawing a Triangle in Raftel Engine

This tutorial will guide you through the steps to draw a simple triangle using the Raftel Engine. This is a fundamental skill necessary for any graphical application development.

Prerequisites

Before beginning, ensure you have the Raftel Engine installed on your system and have a basic understanding of C++ programming.

Steps to Draw a Triangle

Step 1: Setup the Window and Graphics Context

First, include the necessary headers and initialize the window system and graphics context.

#include "raftel/window.hpp"
#include "raftel/shape.hpp"

int main(void) {
  auto windowSystemOpt = Raftel::WindowSystem::make();
  auto windowOpt = Raftel::Window::make("WindowTest", *windowSystemOpt);
  if (!windowOpt) {
      std::cerr << "Error creating window\n";
      return -1;
  }
  windowOpt->MakeContextCurrent();
}

Step 2: Define Triangle Vertices

Define the vertices for your triangle. These coordinates will form the corners of the triangle.

std::vector<float> triangleVertices = {
   0.0f,  0.5f,  0.0f, // Top vertex
   0.5f, -0.5f,  0.0f, // Bottom right vertex
  -0.5f, -0.5f,  0.0f  // Bottom left vertex
};

Step 3: Render the Triangle

Create a shape object with the vertices and draw it in a loop until the window closes.

Raftel::Shape triangle(triangleVertices);
while (!windowOpt->ShouldClose()) {
  windowOpt->clear();
  triangle.draw();
  windowOpt->swapBuffers();
}

Conclusion

By completing this tutorial, you have learned how to set up a basic rendering window and draw a simple triangle using Raftel Engine. These skills form the foundation for creating more complex graphics applications.
