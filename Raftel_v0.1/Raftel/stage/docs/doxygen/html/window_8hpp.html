Detailed Description

Provides the definition of window management and system setup for the Raftel engine.

Date
2025-03-10
Authors
Carlos Mazcuñán Blanes, Marc Folgado Balbás
Copyright
All rights reserved. This file includes classes and methods that handle window creation, context management, input handling, and fullscreen toggling for the Raftel engine. The WindowSystem class is responsible for initializing and managing the GLFW windowing system, while the Window class manages individual windows, allowing for operations such as resizing, context switching, and buffer swapping.

The file also includes essential OpenGL setup such as GLEW initialization and debug message callback setup to facilitate debugging and error handling during OpenGL operations.
